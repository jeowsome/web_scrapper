Success or fail?

Define a function check_success(url) that, given a URL, sends out a GET request and returns "Success" if it succeeds and "Fail" otherwise.

Tip: Remember that the status codes between 200 and 399 indicate that the request succeeded, while all the codes starting with a 4 signalize an error.
